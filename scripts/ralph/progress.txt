## Codebase Patterns
- Lesson pages follow pattern: src/pages/Lesson{N}.jsx
- Lesson sections in: src/sections/lesson{N}/*.jsx
- Content data in: src/data/lesson{N}Content.js
- Quiz data in: src/data/quiz{N}Questions.js
- All animations use Framer Motion
- Styling uses Tailwind CSS with glassmorphism effects
- Progress saves to Supabase

- Dashboard lesson titles are in the `lessons` array in Dashboard.jsx
- `isLessonUnlocked` function controls which lessons are clickable
- TimelineNode component in DashboardGraphics.jsx handles lock icons automatically based on isUnlocked prop
- Each lesson needs its own Context for progress tracking (e.g., Lesson2Context.jsx)
- App.jsx uses state-based navigation (`currentPage` state), not React Router
- Section components receive `onComplete` and `onBack` props
- Export sections via index.js for cleaner imports
